About this guide

\( \LaTeX \) is a typesetting program that helps many academics "typeset mathematical notation easily and legibly". In a way, it is similar to markup languages like Markdown, where we use various symbols to make it do what we want it to do.

This getting started guide on \( \LaTeX \) is mostly based on my notes of some of the videos in the video series titled "Just Enough \( \LaTeX \) to Survive" by Aaron Wong, Ph.d., which is uploaded on this YouTube channel called The Math Repository. This getting started guide will mainly focus on formatting text and writing math notation in \( \LaTeX \).

I can't guarantee the accuracy of this guide, as I'm a beginner. Nevertheless, I hope you find these notes to be helpful as a quick reference after watching the video series.

Thank you for checking this page out and enjoy!

Document structure

A \( \LaTeX \) document is made up of 2 main parts, which is the Preamble and Document environment.

Preamble

To me, this is similar to the <head> section of a webpage. You can type in several parameters of the document here. Some basic parameters are:

  • \documentclass{...} specifies the type of document, and will load features according to their type. Common document types include:
    • \documentclass{article}, which is used for general document making and note-taking.
    • \documentclass{beamer}, which is used to make presentation slides.
  • \usepackage[...](...) allows you to load extra \( \LaTeX \) commands. Fundamental ones are:
    • The geometry package, which allows for margin modification. For example, \usepackage[margin=1.25in]{geometry}.
    • The amsmath and amssymb packages, which are maintained by the American Mathematical Society (AMS). amsmath has a lot of useful functionality and is pretty much used by everyone. amssymb adds several more useful math symbols.
  • \usepackage{graphicx} allows embedding .jpg and .png images.

Note: All \( \LaTeX \) packages are free to use and are already installed by default if you get a standard distribution or use a website where you can create \( \LaTeX \) documents (e.g. Overleaf).

Document environment

The document environment is started with \begin{document} and ended with \end{document}. It is located after the preamble, and contains all of the main contents of the document. 

% This section is the preamble
  
\documentclass{article}
  
\usepackage[margin=1in]{geometry}
\usepackage{amsmath, amssymb}
  
  
% This is where the document environment begins
  
\begin{document}
  ...
\end{document}

Font sizes

First, we can modify the font size of the entire document by adding optional arguments in the square brackets before the curly brackets of \documentclass[]{}, e.g. \documentclass[10pt]{article}. This will change the normal font size of the document, and adjust the other font sizes (small, large, etc.) relative to this.

To modify font sizes, you can use these commands listed below, which are ordered from smallest to largest:

  • \tiny
  • \scriptsize
  • \footnotesize
  • \normalsize
  • \large
  • \Large
  • \LARGE
  • \huge
  • \Huge

These commands can be used in either two ways:

  1. Using it as a switch: You can simply include the command, and everything that follows it will use that font size until you switch it to something else. This is good for long stretches of text with such font size. Otherwise, use the second method...
  2. Using curly brackets to section areas for limited scope: You can also limit the scope of this switch by enclosing it along with the following text that is supposed to be affected by it using curly braces ({}). Note that this method is consistent with the way other "switches" generally work in \( \LaTeX \). 
% 1. Using it as a switch
Normal size. \LARGE Switched! \normalsize Switched back.

% 2. Using curly brackets to section areas for limited scope
Only {\LARGE this} is large!
\[ \text{Normal size. } \LARGE \text{Switched!} \normalsize \text{ Switched back.} \] \[ \text{Only } \LARGE \text{this} \normalsize \text{ is large!} \]

If you need more font sizes, use these packages: extsizes, anyfontsize. However, in most cases, the default font sizes available should be enough.

Font styles

Font styles can be switched using the below functions:

  • Bold text using \textbf{...}
  • Italic text using \textit{...}
  • Underline text using \underline{...}
% Bold text
\textbf{I'm bold}

% Italic text
\textit{I'm italic}

% Underline text
\underline{I'm underlined}
\[ \textbf{I'm bold} \] \[ \textit{I'm italic} \] \[ \underline{\textrm{I'm underlined}} \]

Notice that instead of changing the font style(s) using switches like when changing font sizes, these act more like functions, which apply their actions to the text inside of the curly braces directly following it.

This is a good time to note that a lot of things in \( \LaTeX \) involves the use of brackets. All brackets that are opened must be closed, or not it will throw an error when compiling.

As a tip, since finding missing opening and/or closing brackets can be difficult, you may want to compile your document more often so that there is less areas to check where the error could happen.

You should also be careful when using underlines. Poor spacing and overflow can occur with long lines of underlined text as as \( \LaTeX \) doesn't know how to break such text into multiple lines. To avoid this, use the ulem package by including \usepackage{ulem} in the preamble, which not only fixes text-wrapping issues but provides additional underlines. You also need to use the package's notation:

  • Regular underline using \uline{...}
  • Double underline using \uuline{...}
  • Wavy underline using \uwave{...}

Unfortunately, MathJax (which is used to render the LaTeX math on this page) doesn't support adding packages, so it's not possible to render what these will look like here using it. 

% Regular underlined text
\uline{Regular underlined text}

% Double underlined text
\uuline{Double underlined text}

% Wavy underlined text
\uwave{Wavy underlined text}

Text emphasis

If font styles are used to emphasize text, it's better to use the \emph{...} function, e.g. when writing definitions and theorems (theorem environments are discussed later). It will emphasize the text based on the context of the surrounding text too, e.g. non-italics if everything is italics, etc.

Note that the ulem package changes the way \emph{} emphasizes text, that is by underlining it instead of italicizing it. To switch to the normal mode, type in \normalem anywhere after the ulem package has been called. To switch back to the mode the package provides, type in \ULforem. 

% Preamble
\documentclass{article}

\usepackage{amsmath, amssymb}
\usepackage{ulem}
\normalem % Switches back to the normal emphasis mode.


% Document starts here
\begin{document}

After using the command, text which is emphasized like \emph{this} will now be italicized instead of underlined!

\end{document}

Font families

The default font family used is the Roman text, however you can change it into other font families as shown below:

  • Default (Roman) text: \textrm{...}
  • Sans-serif text: \textsf{...}
  • Typewriter text: \texttt{..}
\[ \textrm{Default (Roman) text} \] \[ \textsf{Sans serif text} \] \[ \texttt{Typewriter text} \]

Note that typewritter text is a monospace font and hence mostly used for code snippets. Also, notice that these are applied as functions, not as switches.

Text alignment

By default, text is justified, where the text is spaced out to fill the full width of the page, leaving straight left and right margins. However, you can align text in the standard different ways too:

  • Center justified text: \begin{center} ... \end{center}
  • Left justified text: \begin{flushleft} ... \end{flushleft}
  • Right justified text: \begin{flushright} ... \end{flushright}

Line breaks in alternate alignment modes

In the non-default alignment methods (i.e. in any alignment mode other than justified), you can add another line of text using a double backslash, and the command [\baselineskip] to add blank lines (to add more, add a 2 in front of the backslash, as shown).

Note that the additional line breaks here are to simulate what the final output will look like, as again, I don't think it's possible to render using MathJax. 

A line of text. \\
Another line of text. \\[\baseilneskip]

Yet another line of text. \\[2\baselineskip]


Final line of text.

Indentation and paragraphs

You can start a new paragraph by adding a blank line of text between each paragraph. By default, indentation will be added to every new paragraph. To remove the indent for a particular paragraph, use the \noindent command at the beginning of the paragraph. 

This is the first paragraph.

\noindent This is a very nice second paragraph without an indent.

This is the third paragraph!

To remove the indent or modify its length for the entire document, you can use the \setlength{\parindent}{1cm} command at the preamble or in the beginning of the document.

Technically, you can use the \setlength{\parindent}{...} command every time you want to do change the indent and then switch it back, however this is not recommended as the document will be awkward flowing.

Units of length

Listed below are the available units in \( \LaTeX \):

  • in: inches
  • cm: centimeters
  • mm: millimeters
  • pt: points; about 1/72 of an inch
  • em: about the width of an m
  • ex: about the height of an x

Math modes

Math modes are another environment with its own rules on how to typeset symbols. The two math modes are:

  1. Display style math: Used when the math is supposed to stand apart from the text, and generally used for long equations and expressions which are being manipulated.
  2. Inline/Text style math: Used when the math is supposed to be part of the text.

Display style math is also stacked more vertically compared to inline math, which can create awkward spacing in paragraphs (don't worry about the summation notation for now). Despite this, you can still force display math in inline mode using \( \displaystyle ... \), and force inline math in display mode using \[ \textstyle ... \].

The below example is copied from the third video in the series titled, "Basic Math Manipulation". 

% Display style Math
\[ \sum_{n=1}^\infty \frac{1}{n^2} \]

% Inline/Text style math
\( \sum_{n=1}^\infty \frac{1}{n^2} \)

% Using display style math in inline mode
\( \displaystyle \sum_{n=1}^\infty \frac{1}{n^2} \)

% Using inline style math in display mode
\[ \textstyle \sum_{n=1}^\infty \frac{1}{n^2} \]

\[ \sum_{n=1}^\infty \frac{1}{n^2} \] \[ \textstyle \sum_{n=1}^\infty \frac{1}{n^2} \]

Display style math

There are two ways to use display style math:

  1. Use backslash (\) and square bracket ([ and ]) combinations as shown below, which to me the backslashes feel like the character used to indicate that you want to do a command. This is commonly used for one-line math. 
    \[ f(x) = (x + 2)^2 -9 \]
    \[ f(x) = (x + 2)^2 -9 \]
  2. Use a display math environment, e.g. the align environment. 
    \begin{align}
  f(x) & = a_2 x^2 + a_1 x + a_o \nonumber \\
       & = x^2 + 4x - 5
\end{align}

% Replace with \begin{align*} and \end{align*} to to disable numbering of equations by default.
    \[ \begin{align} f(x) & = a_2 x^2 + a_1 x + a_o \nonumber \\ & = x^2 + 4x - 5 \end{align} \]
    • By default, all equations are numbered. To prevent one from being numbered, you can add \nonumber to the end of the line or use align* instead of align to not number any equations by default in that environment.
    • Also, the ampersand symbol (&) indicates the location of the alignment, and more than one can be used to align more equations. The below example is also taken from the video. 
      \begin{align*}
2x + 1 & = 9  & 3y - 2 & = -5  & -5z + 8 & = 3 \\
    2x & = 8  &     3y & = -3  &     -5z & = -5 \\
     x & = 4  &      y & = -1  &       z & = -1
\end{align*}
      \begin{align*} 2x + 1 & = 9 & 3y - 2 & = -5 & -5z + 8 & = 3 \\ 2x & = 8 & 3y & = -3 & -5z & = -5 \\ x & = 4 & y & = -1 & z & = -1 \end{align*}

As a tip, the ampersands can be used to organize whitespace in your document, and it is good practice to align the code to the result, so that it's more readable.

Inline/Text style math

There are two ways to display inline math, however the second one is preferred as the first one is dated despite being common.

  1. Use $ to open and close
  2. Use \( to open and \) to close.

Note that when writing variables in a paragraph, you should be in inline math mode, which makes it easy to differentiate text and math. 

The variable \( x \)
The letter x
\[ \text{The variable } x \] \[ \text{The letter x} \]

Arithmetic notation

Finally we get into math notation :D Notice that curly brackets are commonly used in \( \LaTeX \) to group objects so that it treats it as one object, such as in the notation for sums and fractions.

Also, note that math notation must be written inside of one of the math modes to show in the output.

Below is a list of basic arithmetic notation. Please note that for multiplication, do not use asterisks; instead, use one of the multiplication notation provided. Also, for division, be careful to not use bad notation; use fractions instead for the most part.

  • Addition, e.g. 1 + 1 becomes \( 1 + 1 \)
  • Subtraction, e.g. 5 - 3 becomes \( 5 - 3 \)
  • Multiplication (dot notation), e.g. 6 \cdot 4 becomes \( 6 \cdot 4 \)
  • Multiplication (X notation), e.g. 6 \times 4 becomes \( 6 \times 4 \)
  • Division, e.g. 27 \div 9 becomes \( 27 \div 9 \)

Fractions

Fractions can be written out by typing \frac{numerator}{denominator}. It will choose display style or inline style based on the math mode its written in, but you can force either by using:

  • Display style: \dfrac{...}{...}
  • Inline: \tfrac{...}{...}
\frac{\pi}{3}
\[ \frac{\pi}{3} \]

Superscript and subscript

A superscript is small text written above normal text, e.g. to indicate powers. This is done by using a caret (^) symbol after a term.

A subscript is small text written below normal text. this s done by using an underscore (_) symbol after a term.

x^2 \\ % Superscript
a_1    % Subscript
\[ x^2 \] \[ a_1 \]

Grouping using brackets

Note that if there is more than one symbol in superscript/subscript, you need to group them using curly brackets ({}), or not only the first symbol will be in superscript/subscript.

e^{kx} \\ % with grouping
e^kx      % without grouping
\[ e^{kx} \] \[ e^kx \]

Simultaneous superscript and subscript

If both are used at the same time, they will be placed relative to the main character and not each other. Hence, order doesn't matter, but you should use the appropriate one according to mathematical language.

a_1^2 \\ % a sub-1 squared
a^2_1    % a squared sub-1
\[ a_1^2 \] \[ a^2_1 \]

Combining superscript and subscript

Also, by grouping the symbols accordingly (and hence asking LaTeX to treat certain symbols as one object), you can choose whether the superscript/subscript will be stacked or offset. These are mostly stylistic choices depending on each user.

p_1^{a_1} \\   % Stacked
{p_1}^{a_1}    % Offset
\[ p_1^{a_1} \] \[ {p_1}^{a_1} \]

Parentheses

Using the parentheses available on the keyboard will generally make an expression hard to read as they are not of the appropriate size. Example is once again taken from the video.

\[ ( \sum_{n=0}^N ( \frac{1}{a + b} )^2 )^2 \]
\[ ( \sum_{n=0}^N ( \frac{1}{a + b} )^2 )^2 \]

Hence, to let LaTeX to choose the appropriate size, you should use \left( for the left/opening bracket, and \right) for the right/closing bracket. Note that when using this, they must came in pairs so that \( \LaTeX \) can properly calculate the appropriate size.

\[ \left( \sum_{n=0}^N \left( \frac{1}{a + b} \right)^2 \right)^2 \]
\[ \left( \sum_{n=0}^N \left( \frac{1}{a + b} \right)^2 \right)^2 \]

If you need brackets on only one side, you can use \left. and \right. on the side you don't want the parenthesis at.

\[ \left( \sum_{n=0}^N \left( \frac{1}{a + b} \right. \right. \]
\[ \left( \sum_{n=0}^N \left( \frac{1}{a + b} \right. \right. \]

You can also manually choose the size of the bracket using the following modifiers:

  • ( a )
  • \big( a \big)
  • \Big( a \Big)
  • \bigg( a \bigg)
  • \Bigg( a \Bigg)
\[ ( a ) \big( a \big) \Big( a \Big) \bigg( a \bigg) \Bigg( a \Bigg) \]

Writing text in math mode

To write text in math mode, e.g. a condition for an equation, you must use the \text{} command, otherwise LaTeX will interpret it as a product of variables.

\[ n = ab where a and b are natural numbers \]
\[ n = ab where a and b are natural numbers \]

You must also type explicit spaces, since LaTeX will not add them automatically.

\[ n = ab \text{where} a \text{and} b \text{are natural numbers} \]
\[ n = ab \text{where} a \text{and} b \text{are natural numbers} \]

There is more than one way to write the same text, however it's helpful to organize the code in a logical manner, to make the code easier to read. E.g., since the statement is an equation and condition, you may put the condition completely inside of the \text{} command, and use \( ... \) to activate inline math appropriately for e.g., variables, as shown in the second line.

\[ n = ab \text{ where } a \text{ and } b \text{ are natural numbers} \]

% This one may be better:
\[ n = ab \text{ where \( a \) and \( b \) are natural numbers} \]
\[ n = ab \text{ where } a \text{ and } b \text{ are natural numbers} \] \[ n = ab \text{ where \( a \) and \( b \) are natural numbers} \]

Greek letters

Greek letters can generally be written out by writing a backslash followed by the name of the variable. To get the capitalized letter, capitalize the first character of the letter name. Some characters in the Greek alphabet are the same, so there's no need to use the slash (just use whatever is available on the keyboard). There are also variations of the lowercase Greek letters, which you can get by adding "var" before the Greek latter name.

Below is an example with the variable phi

\phi \\     % Lowercase phi
\varphi \\  % Variation of lowercase phi
\Phi        % Uppercase phi
\[ \phi \] \[ \varphi \] \[ \Phi \]

Blackboard font

Blackboard font are letters which have double lines, commonly used to represent important sets in mathematics. These can be entered by using the \mathbb{} command and entering the desired letter into the curly braces.

As examples, below is a list of some common sets with their corresponding letters in blackboard font:

  • Natural numbers: \mathbb{N} becomes \( \mathbb{N} \)
  • Integers: \mathbb{Z} becomes \( \mathbb{Z} \)
  • Rational numbers: \mathbb{Q} becomes \( \mathbb{Q} \)
  • Real numbers: \mathbb{R} becomes \( \mathbb{R} \)
  • Complex numbers: \mathbb{C} becomes \( \mathbb{C} \)
